This is a python program when ran, will open a window that allows you to insert, update and delete rows as well and change tables.

To insert data click on the insert button and enter the data when done click insert. Strings must have '' (single quotes) encasing them.
To update a row click on modify follow the prompts, user must always enter the primary key in all cases an ID number to modify the row.
To delete user simply must click on the delete button then enter the ID number IF the row has a child row it will not be deleted.
to change tables simply click on the change button then enter the name of the table
